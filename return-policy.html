What is urbanic's Return policy? How does it work?

We're happy to share about the Return policy offered to our customers. With this option, if you don't like the product or if there is any issue, you will have the option of requesting the return request within 10 days (Majority of items are returnable, with the exception of lingerie, jewelry, holiday accessories and some surfer swimsuits. It varies from product to product so please check the product details page for more information) from product delivery. At this moment, we are not able to facilitate the exchange of the products. The only requirement of the return policy is that the item you're returning should be in its actual conditions. Tags and packaging should remain intact and the dress should not have been used. Once the item is received at the warehouse, there will be a quality check post which the refund will be processed. You're more than welcome to try on the product; however, please take adequate measure to avoid any damage to the product.

You can now apply for the return request directly from your Urbanic account in the app or m-site. Please follow the simple steps mentioned below to submit your return request within 10 days from the order is delivered

  1. Login to your account on Urbanic which you have placed the order app from which you have placed the order
  2. Go to "Me" section from the home page
  3. From the order list, select the order to apply for return you want to apply for return
  4. In the return application, select the products and quantity you want to return. In case of multiple products, please apply for return for each item by clicking on the return option against the item.
  5. Select reason for return and write some details about the issue you have faced
  6. Upload images for our team to review your request
  7. Provide required details like pickup address, expected time for pickup and bank details in case of COD order
  8. Submit the request

If you are not able to submit the return request online, please reach out to our Chat Team from your app or email our support team at support@urbanic.com

Once the return request is successfully submitted, our team will review your request and once the request is approved, our courier partner will be aligned to pick up the order as per your prescribed date and time, subject to availability. The refund will be processed when the product is delivered to our Warehouse and meets the quality parameters of the item. There will be no refund issued if the received product does not pass the quality check.

Please note, the amount will be refunded for the items only, other fees like shipping fee, cod service charges etc, are non-refundable. Also, our shipping partner will pick up your returned items at no extra as of now.

There are some exceptions in return policies of some times

The following EXCEPTIONS and RULES apply to this Policy:

  1. Some products like fine jewelry, watches and selected products which are susceptible to damage can only be returned for a limited number of days. Please read the Product Detail Page to see the number of days up to which a product can be returned once delivered.
  2. All items that need to be returned must be unused and in their original condition with all original tags and packaging intact.
  3. In case you have purchased an item, which has a free gift/offer associated with it and you wish to return the main item, then your gift item also needs to be returned in order to process the required refund.
  4. Urbanic will not be liable for the products returned by mistake. If an extra or a different product is returned by mistake, Urbanic would not be accountable for misplacement or replacement of the product and is not responsible for its delivery back to the Customer
